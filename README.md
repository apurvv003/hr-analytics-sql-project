# HR Analytics SQL Project
## Overview
This project analyzes HR employee data using SQL Server.
## Skills Used
- SQL
- GROUP BY
- ORDER BY
- Aggregation Functions
- Filtering
- HR Analytics
- Data Analysis

## Key Insights
- Calculated employee attrition rate
- Identified departments with highest terminations
- Analyzed employee engagement and satisfaction
- Evaluated work-life balance trends
- Found departments with highest employee ratings

## Tools Used
- SQL Server Management Studio (SSMS)
- SQL Server
- Kaggle HR Dataset
